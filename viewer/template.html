Code as navigable hypertext

Pick a function from the outline. Every resolved reference inside it is a link — click it and the definition opens in a card, right here, without losing your place. Links inside a card work the same way, so cards spawn cards.

Tokens are styled by what the indexer could actually prove: a real cross-file link, a local or parameter, a data column (tidyverse NSE), or library code. Nothing is a dead link.
